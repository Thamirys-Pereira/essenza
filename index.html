Projeto final

Objetivo

Criar um e-commerce (Essenza), utilizando HTML, CSS e Javascript. Com o Javascript criar as iterações necessárias para o aplicatiivo web, tais como:


  • Arrays
  • Ojetos
  • Funções
  • Etc

    • A intenção desse projeto é criar um aplicativo web que permita a interação do usuario com o produto, o carrinho de compras, login, e mecanismos de busca.
